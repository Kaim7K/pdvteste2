# Roadmap técnico

## V1 entregue neste pacote

- Login com JWT e cookie HTTP-only.
- Perfis gerente/vendedor.
- Produtos com ou sem código de barras.
- Código interno automático.
- Cadastro rápido.
- Tela de vendas.
- Captura de leitor de código de barras via teclado.
- Carrinho com quantidade e peso.
- Produto com valor errado.
- Pagamento misto.
- Venda fiado.
- Vendas minimizadas salvas no banco.
- Histórico de vendas.
- Recibo em tela e impressão/PDF via navegador.
- Estoque com baixa automática e estoque negativo permitido.
- Auditoria individual e geral.
- Relatórios com cards, gráfico e insights por regra.
- Configurações administrativas.

## Próximas melhorias profissionais

1. Testes automatizados com Playwright e Vitest.
2. Permissões customizadas por usuário além do perfil.
3. Busca automática de imagens por integração externa.
4. Upload real de imagens em Supabase Storage ou Cloudflare R2.
5. Impressão térmica 58mm/80mm.
6. PWA offline com IndexedDB.
7. Sincronização e versão local/Electron.
8. Multi-caixa com controle de sessão de caixa.
9. Fechamento de caixa.
10. NFC-e, se o projeto exigir operação fiscal.
